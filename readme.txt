== Description ==

This plugin allows you to add a shortcode and convert any part of text to image. Pluging is usefull if you want to protect e-mail addresses or links in your post.

To use plugin just install it and activate. When the plugin is activated you can use [e2i] text you wish to convert to image [/e2i]. There are two additional options. If you use [e2i type='admin'] site administrators will be able to see normaln text. Another option is [e2i type='login'] which will allow all logged in users will see normaln text and guests will see image.

== Installation ==

1. Upload plugin to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. In post/page editor add [e2i]...[/e2i] around text you want to convert to image. Also use [e2i type='admin']...[/e2i] or [e2i type='login']...[/e2i] to allow administrators or
logged in user to see normaln text
